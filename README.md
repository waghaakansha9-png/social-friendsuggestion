# 🌐 Social Friend Suggestion System

## 📌 Project Description
The Social Friend Suggestion System is a web application that suggests new friends based on social network connections using graph algorithms.

This project demonstrates how social media platforms recommend friends using mutual connections.

---

## 🚀 Features
- User Login Page
- Friend Suggestion System
- Social Network Graph Visualization
- Modern Web Interface
- Real-time Friend Connections

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Graph Logic: Python Data Structures
- Deployment: Render
- Version Control: GitHub

---

## ⚙️ How It Works
1. Users log into the system.
2. Social connections are stored as a graph.
3. The system analyzes mutual friends.
4. Suggested friends are displayed to users.
5. Graph visualization shows relationships.
